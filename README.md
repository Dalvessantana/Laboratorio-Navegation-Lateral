#Navegation Lateral

Interface de dashboard com menu lateral desenvolvida com foco em navegação intuitiva, organização de conteúdo e design moderno.

---

## Sobre o Projeto

O **LabPanel UI** é um layout de interface inspirado em painéis administrativos, apresentando uma **sidebar de navegação** com estrutura clara e elementos visuais bem definidos.

O projeto prioriza a experiência do usuário através de uma navegação simples e um design limpo.

---


### Área Principal

- Imagem de fundo em tela cheia  
- Aplicação de overlay escuro para contraste  
- Destaque visual para o menu lateral  

---

## Design

O projeto segue uma abordagem moderna baseada em:

- Tema escuro (Dark UI)  
- Efeito translúcido (glassmorphism)  
- Tipografia limpa  
- Ícones minimalistas  
- Estrutura inspirada em dashboards  

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  

---

## Objetivo

Este projeto foi desenvolvido para:

- Praticar criação de layouts com sidebar  
- Trabalhar design moderno (glassmorphism)  
- Melhorar organização visual de interfaces  
- Simular estrutura de sistemas administrativos
